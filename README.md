# Homework 5 - Trail & Timber Landing Page

## Project Description

This project is a responsive landing page based on the provided Trail & Timber prototype. The site uses HTML, Sass, CSS Grid, Flexbox, Sass variables, partial files, nesting, and mixins.

## Assignment Requirements

- Used Sass
- I Included a variable file
- I Included a mixins file
- I Included at least one additional Sass partial file
- Used nested selectors where appropriate
- Used at least one custom mixin
- Used CSS Grid
- Used Flexbox
- Created responsive layouts for desktop, tablet and mobile
- Display a mobile navigation icon instead of the full navigation menu on smaller screens
- Difficulty note to myself - my monitor is huge and the hero seems to always want to over stretch the image even though my code is good.
